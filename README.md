# Stack
As seguintes ferramentas, *frameworks* e bibliotecas foram utilizadas na construção deste projeto:

### 🖥️ Back-end

| Tecnologia | Descrição |
| :--- | :--- |
| **Kotlin** | Linguagem de programação principal. |
| **Spring Boot** | *Framework* para desenvolvimento rápido e robusto de aplicações *backend*. |
| **Spring Security** | Utilizado para garantir a segurança e autenticação da API. |
| **Docker** | Usado para containerização do *backend* e do banco de dados, facilitando a execução em diferentes ambientes. |

### 💻 Front-end

| Tecnologia | Descrição |
| :--- | :--- |
| **Dart** | Linguagem de Programação Principal. |
| **Flutter** | Framework de UI voltado para Design de Aplicações Móveis. |

### ⚙️ Infraestrutura & DevOps

*   **GitHub Actions:** Configurado para *Continuous Deployment* (CD) do *backend* (conforme badge).
*   **Docker Compose:** Utilizado para orquestrar a execução local do *backend* e do banco de dados.
